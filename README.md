# Demo Django + Tailwind CSS

Projeto de demonstração desenvolvido com **Django**, **Tailwind CSS** e **Docker Compose**, utilizando **SQLite** como banco de dados.

A aplicação apresenta uma página inicial moderna, estilizada com Tailwind CSS, além de integração com o painel administrativo do Django para gerenciamento de mensagens.

## Tecnologias Utilizadas

- Python 3
- Django
- Tailwind CSS (via CDN)
- Docker
- Docker Compose
- SQLite

## Funcionalidades

-  Página inicial responsiva
-  Interface moderna utilizando Tailwind CSS
-  Listagem de mensagens armazenadas no banco SQLite
-  Painel administrativo do Django
-  Ambiente containerizado com Docker Compose
  
## Inserindo mensagens

As mensagens podem ser cadastradas diretamente pelo painel administrativo do Django.

Após o cadastro, elas aparecerão automaticamente na página inicial.


##  Objetivo

Este projeto foi desenvolvido para fins educacionais, demonstrando:

- Estrutura básica do Django
- Utilização do ORM
- Templates
- Dockerização de aplicações
- Estilização com Tailwind CSS
- Integração com SQLite

Projeto desenvolvido para fins acadêmicos.
